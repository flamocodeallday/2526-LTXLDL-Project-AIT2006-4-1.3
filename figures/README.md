# Figures

This directory stores all visual outputs generated during the analysis, organized by month.

All subfolders and figures are created automatically when the analysis notebooks are executed.

---

## Directory Structure

- `January_figures/`
- `February_figures/`
- `...`
- `December_figures/`

Each monthly subfolder contains figures related to:
- Trip volume trends
- KPI visualizations
- Temporal and zone-level analysis results

---

## Notes

- This directory contains **generated figures only**
- Subfolders are created dynamically during execution
- Figures may be overwritten when the analysis is re-run
- Files should not be edited manually

---

## Reproducibility

To regenerate all figures:
1. Ensure processed data is available
2. Run the analysis notebooks
3. All figures will be recreated automatically in this directory
